# Reference Design

This OpenSpec change is anchored to the requester-provided reference design image:

```text
583f8df5-e2f5-4076-ada1-b649c459d557.png
```

The image is the visual north star for the initial UI-only redesign.

Key direction:

- primary purple `#6d49f4` and secondary teal `#00978a`;
- clean white SaaS landing page;
- refreshed Resubuild logo with document, rebuild/upward arrow, and subtle AI/spark motif;
- compact app-icon / favicon variant;
- line icons for import, edit, AI/sparkle, export, shield, lock, check, list/table, CV card, and application workspace;
- hero headline: `Import your PDF CV. Get a polished CV in seconds.`;
- product mockup showing the CV editor/preview;
- feature cards for AI PDF Import, Clean Editor, Job Tailoring, and One-Click Export;
- three-step flow: Import your CV, Review & edit, Tailor & export;
- benefits row: No watermarks, Structured JSON Resume, Your data is private, ATS-friendly exports;
- authenticated workspace preview showing My CVs cards, recent applications table, status badges, and existing CTAs;
- purple/teal final CTA band and refreshed footer.

The image should guide reusable components, tokens, spacing, gradients, iconography, and layout rhythm. It should not be copied as a fixed static screenshot.

This phase remains UI-only. Advanced analysis surfaces, comparison views, recommendation queues, and new AI metadata contracts are outside this change.
